总结

apply 和 call 均可以改变上下文执行环境，apply 可以动态传入数组参数， call 可以按照正常调用函数的方式执行函数，各有特点， apply 配合数组、对象还可以巧妙的玩出各种骚操作，增加效率和乐趣！
