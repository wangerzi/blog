#### 常用命令组合

##### awk

```shell
awk 'BEGIN{} {} END{}'
-F 指定分隔符
NR 行数
```

作用：循环每行内容，并处理每行信息，多用于统计 `test.txt` 文本内容

```text
1,aaa
2,bbb
3,ccc
```

获取 `,` 分隔的第二列的内容

```shell
awk '{pint $2}' -F, test.txt
```

获取第一列所有数字之和以及平均值

```shell
awk 'BEGIN{sum=0} {sum+=$1} END{print sum}' -F, test.txt
awk 'BEGIN{sum=0} {sum+=$1} END{print sum/NR}' -F, test.txt # NR 指行数
```
